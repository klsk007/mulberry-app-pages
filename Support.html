Mulberry Support

Mulberry is an iPad handwriting AI notebook. This page summarizes common troubleshooting steps for App Store review and users.

Handwriting Recognition

If recognition is inaccurate, try writing larger, using higher contrast ink, or changing the recognition language in Settings.

Model Connection

The app uses a default test model connection and also supports user-configured compatible model services. If answers do not appear, check Settings and confirm the API key, base URL, and model name.

Voice Features

Voice input requires iOS microphone and speech recognition permissions. Permissions can be changed in the iOS Settings app.

Pro Purchase

Mulberry Pro is a non-consumable lifetime unlock purchased through the App Store. If Pro does not unlock after purchase, open the purchase screen and tap Restore Purchases.

Contact

Please use the support contact shown on the App Store product page.
