Morus Paper Support

Morus Paper is the in-app brand for Magic Spellbook AI / 魔法书AI笔记, an iPad handwriting AI notebook. This page summarizes common troubleshooting steps for App Store review and users.

Handwriting Recognition

If recognition is inaccurate, try writing larger, using higher contrast ink, or changing the recognition language in Settings.

Model Connection

By default, the app uses iOS on-device OCR and Apple's on-device language model, so no API key is needed on supported devices. You can explicitly select Offline Llama or High-quality API in Settings; the app does not silently switch models.

Offline Model Storage

Offline Llama is an optional App Store download of about 1.9 GB and is not part of the base install. Select it under Answer Model to download it. Switch to Apple On-device or High-quality API, or tap Release Offline Model, to let iPadOS reclaim the cache. The final purge time is managed by iPadOS.

Voice Features

Voice input requires iOS microphone and speech recognition permissions. Permissions can be changed in the iOS Settings app.

Pro Purchase

Morus Paper Pro is a non-consumable lifetime unlock purchased through the App Store. If Pro does not unlock after purchase, open the purchase screen and tap Restore Purchases.

Contact

Open a support request. Please do not include API keys, private diary text, or other sensitive information.
